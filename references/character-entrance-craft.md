# Character Entrance Craft / 人物出场技艺

> 这是 `hard-rules.md` 第 1 条的细化展开。任何重要人物、命名人物、反复出现的人物**第一次出场**时必须按本文件操作。
> Detailed expansion of Hard Rule #1. Apply on every named or recurring character's first appearance.

---

## 核心原则 / Core Principle

人物出场不是"通报名字"，是给读者一个**入场落点**——一个能记住、能预期、能误读、能投射欲望或恐惧的具体存在。

A character entrance isn't an announcement. It's a landing — something the reader can remember, anticipate, misread, and project desire or fear onto.

---

## 5 锚点最低门槛 / 5-Anchor Minimum

任何重要/命名/反复出现的新人物，第一次出场必须命中至少 4/5 个锚点。

### 锚点 1：身份锚点 / Identity Anchor

他**是谁**？职位、关系、阵营、社会位置、当前场景的功能。

避免泛标签（"一个男人"、"一位老者"），给具体定位（"她姐夫的下属"、"夜班护士长"、"东宫旧人"、"刚被收编的散兵营队长"）。

### 锚点 2：外貌/物理锚点 / Visual or Physical Anchor

至少**一个具体细节**：脸、身形、年龄感、衣着、姿态、声音、动作、气味、伤痕、随身物件。

**禁止**只用泛词（漂亮、冷峻、高大、儒雅、危险、神秘）。**要**用可视化、可记忆、能反复呼应的具体物。

例：
- 弱（不行）："他长得很英俊。"
- 强（行）："他左手腕上一圈晒痕——常年戴表的人摘了表才会有的那种白。"

### 锚点 3：场景关联锚点 / Scene Relevance Anchor

为什么**此刻**他在这里？和当前冲突什么关系？读者需要立刻知道他对当下场景的"重量"。

避免"角色随机经过"。每个新人物的出场地点和时机都该是被剧情召唤来的。

### 锚点 4：POV / 社会阅读锚点 / POV or Social Reading Anchor

主角或旁人**如何理解、误解、忌惮、轻视、尊敬、欲望、排斥**他？

这是把"人物"变成"人物对 POV 的意义"的一步。

第一人称：必经叙述者偏见、记忆、情绪、欲望过滤。
第三人称：通过近距离感知、旁人反应或场景压力建立——**不要**冷冰冰镜头平铺。

### 锚点 5：行为钩子 / Behavioral Hook

**一句话、一个动作、一个习惯、一个选择、一个压力反应**，让这个人不只是标签。

例：
- "她进门没看任何人，先把窗帘拉开半指，让外面街灯落进来一道。"
- "他听完笑了一下，但那个笑没到眼睛，停在嘴角就掐灭了。"
- "对方递来的茶他没接，等对方放下，又自己端起来——多走了一秒，姿态没让出去。"

## 第一人称 vs 第三人称 / POV Filtering

### 第一人称
人物介绍**必须**经过叙述者的：
- **偏见**（他原本就讨厌这类人 / 他被某段经历训练得对某种特征过敏）
- **记忆**（他想到曾经认识的另一个人 / 一个旧场景被触发）
- **情绪**（紧张、欲望、嫉妒、警戒、安心——选择"看到什么"和"看漏什么"）
- **欲望**（叙述者想要从这个人身上得到什么；读者通过 POV 的"看见"理解 POV）

错误示范：
> "他叫陆景行，三十二岁，是 X 公司副总。我看着他英俊的脸，心里有些紧张。"

正确示范：
> "陆景行——这名字我两年前在父亲书房听过一次。他比我想象的瘦。袖口是没烫平的痕迹，跟传言里那个谨慎得过分的男人对不上。我的紧张不是因为他长得英俊，是因为他没有一秒在打量我。"

### 第三人称（近距离 / 中距离）
通过：
- **近距离感知**（POV 角色注意到的、忽略的、回避去想的）
- **旁人反应**（场内其他人怎么调整姿态、打招呼、避开视线、降声音）
- **场景压力**（他一进来空气怎么变了；之前的话题为什么停了）

避免**冷镜头平铺**——不要写得像演职员表："这是 X，X 岁，职业是 X，外貌是 X。"

## 重要人物的人物卡叠加 / Stable Anchors + Current Overlays

重要人物建立**稳定外貌锚点** + **当前状态叠加**。后续描写禁止反复用泛词。

### 稳定锚点（卡上常驻）
- 脸 / 眼 / 身形 / 姿态 / 动作习惯 / 声音 / 气味 / 标志性物件或痕迹
- 默认服饰风格 / 颜色 / 材质偏好 / 身份信号

### 当前状态叠加（每次出场更新）
- 受伤 / 战损 / 疲惫
- 伪装 / 礼服 / 公开场合形象
- 贫穷展示 / 富贵展示
- 权力地位变化（晋升、失势、流放、归来）
- 情绪变化（愤怒、悲恸、压抑、释然）

### 禁词检查（描写重复的红旗）
连续两次以上用了下列词且没有更具体细节支撑——重写：
- 漂亮 / 美丽 / 好看 / 英俊
- 冷峻 / 冷漠 / 高冷
- 高大 / 修长 / 挺拔
- 儒雅 / 温润 / 优雅
- 危险 / 神秘 / 锋利
- 强大的气场 / 不容忽视的存在感

替代方案：用**可视化的具体物**、**可记忆的姿态**、**他人的具体反应**。

---

## 走过场 / 功能性人物 / Walk-on Characters

不需要 5 锚点，但**至少**给：
- 角色（"门房"、"卖花的小贩"、"邻桌的女学生"）
- 一个视觉/行为细节（**一个**就够，但不能没有）
- 当下相关性（他为什么在这一句话里出现）

例：
> "门房没抬头，只把登记本往他面前推了推。指甲缝里有黑色——油渍，不是脏。"

---

## 出场后的复现规则 / Re-Appearance Rules

人物第二次、第三次出场时**不要**重复第一次的所有描写。规则：

1. **复用稳定锚点**（一两个，提醒读者）
2. **更新当前叠加**（伤、累、新衣服、新身份、新情绪）
3. **加入新发现**（POV 第一次没注意到的细节，或对方的新行为）
4. **通过他人反应折射**（不只是 POV 看，让其他角色对他的反应也变成描写）

## 章节包字段 / Packet Fields

章节包"新人物出场"段必须按下列模板填齐（已在 `hard-rules.md` 章节包模板中）：

```
## 新人物 / New Character Entrances
- 姓名：
  - 1 身份锚点：
  - 2 外貌/物理锚点（具体物，禁泛词）：
  - 3 场景关联锚点（为什么此刻在这里）：
  - 4 POV/社会阅读锚点（POV 如何理解他）：
  - 5 行为钩子（一句话/动作/习惯/选择）：
```

## 审计问题 / Audit Questions

- 新人物 5 锚点是否至少齐 4 项？
- 第一人称是否经过叙述者偏见、记忆、情绪、欲望过滤？
- 第三人称是否通过近距离感知、旁人反应或场景压力建立？还是冷镜头平铺？
- 重要人物有没有可记忆的、不重复的外貌或行为锚点？
- 是否反复用泛词（漂亮/冷峻/高大）而无具体细节支撑？
- 走过场人物是否至少有"角色 + 一个细节 + 相关性"？
- 复现时是否更新了当前状态叠加，而不是重复第一次的描写？

### P0 必修
- 新出场重要/命名人物只有名字或职位，5 锚点中命中不足 4 项
- 第一人称下叙述者完全没有过滤（像观察笔记）
- 走过场人物只有一个名字或职位，无任何细节

### P1 应修
- 重要人物连续 2 章只用泛词描写
- 复现时机械重复第一次的描写、不更新当前状态
- 第三人称冷镜头平铺超过约 200 字

## 实例对比 / Worked Examples

### 例 1：女主第一次见到男主（第一人称）

**弱（违反 P0）**：
> 我抬起头，看到一个英俊的男人走进来。他叫顾承珩，是顾家的二少爷，今年二十六岁。他冷冷地看着我，气场强大。

问题：5 锚点只命中 1（身份）；外貌全是泛词；POV 完全没过滤；行为钩子缺失。

**强**：
> 我抬起头的时候，他已经站在门里。比我想象的高半个头，大衣下摆还在抖——外头风大，他没让司机送进来。我先注意到的不是脸，是他左手——空着，右手提着一个旧公文包，那种边角磨白了的牛皮，跟他这身衣服不搭。门房在他进来时把登记本收了，没让他签。所以这就是顾承珩。两年前我父亲在饭桌上骂过的那个人。他没看我，先看了我身后的窗——确认了什么，才把视线落下来，落在我手里那杯没动过的茶上，停了半秒。

5 锚点：身份（顾承珩，父亲骂过的人）；外貌（高半个头、大衣下摆、左手空、旧公文包磨白牛皮）；场景关联（门房没让签——他不是普通客人）；POV 阅读（我父亲骂过他、我注意到他先看窗）；行为钩子（先确认窗、视线落在没动的茶上）。

### 例 2：配角进场（第三人称）

**弱**：
> 这时一个老者走了进来，他是太医院的院判，姓王，医术高明。

**强**：
> 帘子从外被掀起一角，先进来的是药味——苦的，带着点土腥气，是给老人煎的方子才会用的引子。然后是人。王院判七十出头，半个肩膀比另一半矮了寸许，是常年伏案号脉留下的。他进门没行礼，先看了一眼床上的人，眉头先皱了，才转向皇后。

5 锚点：身份（太医院院判、看病人的）；外貌（七十、一肩低半寸、伏案号脉留下的）；场景关联（被叫来给床上的人看病）；社会阅读（不行礼——身份够老或场合够急）；行为钩子（先看病人、再看皇后——专业优先于礼节）。

---

## 北极星 / North Star

> 读者读完一个人物的出场，应该能在脑子里**画出一个具体的人**，对他**有期待或警戒**，并且**知道他和当前剧情有什么关系**——不是收到一份人物档案。

EN: After reading an entrance, the reader should be able to picture a specific person, hold expectation or wariness about them, and understand their stake in the current scene — not receive a character file.
